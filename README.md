# Simple Web Calculator

A clean, responsive, and lightweight web-based calculator built using pure HTML5, CSS3, and JavaScript. This project handles basic mathematical computations dynamically using inline events and JavaScript logic.

## 🚀 Features

- **Basic Arithmetic Operations:** Supports addition (`+`), subtraction (`-`), multiplication (`*`), and division (`/`).
- **Real-time Display:** Updates immediately as buttons are pressed.
- **Responsive Layout:** Designed with modern CSS Flexbox wrapper to ensure buttons wrap neatly within the calculator card.
- **Decimal & Clear Support:** Includes a decimal (`.`) precision button and a quick Clear (`C`) button to reset the state.

## 🛠️ Tech Stack Used

- **HTML5:** For structured input element (`readonly` display) and functional calculator buttons.
- **CSS3:** Flexbox layout (`flex-wrap`), custom sizing, modern borders, and centered positioning.
- **JavaScript (ES6):** State management using `Current_Display`, direct DOM queries (`document.querySelector`), and dynamic evaluation via `eval()`.
